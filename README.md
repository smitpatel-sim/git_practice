# Merge Conflict Practice

Use this repository to practice resolving merge conflicts.

## Practice Steps

1. Stay on `main`.
2. Run `git merge conflict-01`.
3. Run `git merge conflict-02`.
4. Resolve the conflicts in the five files.
5. Commit the merge, then continue with more branches.
